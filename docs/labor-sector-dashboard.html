Method Notes

NFP monthly and YTD changes are computed from CES employment levels in thousands. YTD equals latest level minus prior December. JOLTS openings are a stock, so YTD change is latest minus prior December. JOLTS hires, quits, and layoffs/discharges are flows, so YTD is cumulative year-to-date. ADP is optional because the official public ADP release does not provide a stable BLS-like historical sector API. Revision basis: BLS current vintage / revised history. This is not a real-time pre-revision vintage dashboard.
